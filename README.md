## Installation
To set up the portfolio locally on your machine:
1. Clone the repository:
   ```bash
   git clone https://github.com/Flora72
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your browser to view the portfolio.
